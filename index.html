Welcome to FizzBuzz!

  • Generate a pair of numbers between 2 and 9: Fizz and Buzz
    • The two numbers Fizz and Buzz will not be the same as each other
    • Fizz and Buzz remain for duration of the game
  • Generate a set of target numbers between 10 and 100
  • Work out whether your Target Number can be divided by:
    • Fizz
    • Buzz
    • Neither number
    • Or both numbers, in which case it's FizzBuzz!

Scoring system

  • If you tap Fizz, Buzz or No! correctly: 1 point
  • If you tap Fizz or Buzz when the answer was FizzBuzz: 1 point
  • If you tap FizzBuzz correctly: 2 points
  • Anything else: 0 points

Example Game

In this example game, Fizz = 3, Buzz = 5

  • Your first Target Number = 6: you tap Fizz
  • Your next Target Number = 10: you tap Buzz
  • The next Target Number = 13: you tap No!
  • The next Target Number = 15: you tap FizzBuzz
